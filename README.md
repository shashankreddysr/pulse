# Pulse Video Sensitivity Dashboard

Full-stack real-time video sensitivity management platform.

## Live Demo
Frontend: https://pulse-q360n35ze-shashanks-projects-24e54bb4.vercel.app/
Backend API: https://pulse-7nyf.onrender.com

## Tech Stack
Frontend: React + Vite
Backend: Node.js + Express
Database: MongoDB Atlas
Deployment: Vercel + Render

## Features
- Authentication system
- Real-time streaming architecture
- Secure uploads
- Multi-tenant support
- Audit logging
- API health monitoring

## Setup

Frontend:
cd frontend
npm install
npm run dev

Backend:
cd backend
npm install
npm run start
